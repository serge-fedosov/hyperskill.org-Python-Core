Description

Great! You passed the first step and got the cut plasmid. To make a bacterium glow, the green fluorescent protein (GFP) gene must be inserted into the bacterial plasmid.

The second step starts with double-stranded GFP (i.e., GFP composed of the original and complementary strands). Since you are already familiar with the principle of complementarity, we ask you to refresh your knowledge here (see point 1 in the figure).

Objectives

At this stage, you need to get the double-stranded GFP gene. Use your knowledge to compose a function that automates this action.

Your program should:

  • Input a sequence of an original strand;
  • Create a complementary strand;
  • Print the original and complementary strands, separated by a newline character (\n).
Examples

The greater-than symbol followed by a space (> ) represents the console input.

Example 1:

> CTAGAGGATCCCCG
CTAGAGGATCCCCG  # the output starts from this line
GATCTCCTAGGGGC

Example 2:

> ATGAAACAGCATGACTTTTTCAAGAGTGCCATGCCCGAAGGTTATGTA
ATGAAACAGCATGACTTTTTCAAGAGTGCCATGCCCGAAGGTTATGTA  # the output starts from this line
TACTTTGTCGTACTGAAAAAGTTCTCACGGTACGGGCTTCCAATACAT
